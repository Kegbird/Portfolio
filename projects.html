Order Please

Order Please is a management application developed for my family (my family runs a restaurant); it's composed from 4 parts, all using a database in which are stored dishes, drinks and all other things that can be sold to our customers. These 4 parts are:

  • An android application
  • A db management desktop application
  • A desktop application which simulates a cashdesk
  • A small web service which espose db data
The android application is used by waiters allowing them to take orders from customers: all orders taken are received by the cashdesk application that stores all item ordered from each table in the service room. The cashdesk application receives all order: each order maybe will contains different voices, each one with a different category, so the application splits the received order into many, printing them on different printers; for instance, all drinks are redirected to a printer in the bar, all dishes into the kitchen's one etc. etc. Before the customer leaves, the application prints a brief resum of all dishes and drinks ordered by him/her.

Spline Ramp

Spline Ramp is a simple puzzle game developed with Unity and built as webgl game; the main objective of the game is to create a ramp that allows the player to launch a ball into a net. Each ramp is described by a mathematical definition of a certain curve (Bezier, Hermite, BSpline).

Filters comparison

The purpose of this project was to compare different parallel implementations of notorious computer graphics filter, such as Canny, Module and Robert, with their classic sequential version. All parallel implementations exploits and combine different features offered by NVIDIA GPU, throught the framework CUDA.
